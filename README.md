## URL Shortener App

### *URL shortener app using tkinter:*
``` python
from tkinter import *
```
#### Example:
``` python
import pyshorteners
s = pyshorteners.Shortener()
short_link = s.tinyurl.short('https://aparat.com')

out_put = https://tinyurl.com/y5ww6vjc
```
#### *VPN must be turned on for shortening url*
